## はじめに

このシステムは、カットモデル画像を安全に管理・共有するためのWebアプリケーションです。
画像の利用には承認が必要で、ダウンロードした画像には透かしが埋め込まれます。

### 対応ブラウザ
- Google Chrome（推奨）
- Microsoft Edge
- Safari
- Firefox

---

## ログイン

### 手順
1. システムのURLにアクセス
2. 「Microsoftアカウントでサインイン」をクリック
3. 会社のMicrosoftアカウントでログイン

> **注意**: 事前に管理者によるユーザー登録が必要です。登録されていないアカウントではログインできません。

## 画像を閲覧する

### フォルダ構造
画像はフォルダで整理されています。フォルダをクリックして中身を確認できます。

### パンくずリスト
現在の位置が画面上部に表示されます。クリックで上位フォルダに戻れます。

```
ルート > カテゴリA > サブカテゴリ1
```

### 画像プレビュー
- 画像をクリックすると拡大表示されます
- 左右の矢印キー、または画面端のボタンで前後の画像に移動できます
- スマートフォンではスワイプで移動できます

### 権限レベル
画像には3種類の権限があります：

| 権限 | 説明 |
|------|------|
| **閲覧のみ** | 画像を見れますが、ダウンロードには申請が必要です |
| **ダウンロード可** | 申請なしで直接ダウンロードできます |
| **編集・削除可** | 画像の編集・削除ができます（通常は管理者のみ） |

## 画像の利用申請

「閲覧のみ」の画像をダウンロードするには、利用申請が必要です。

### Step 1: 申請内容の入力

1. 画像をクリック
2. 「画像の利用申請」モーダルが表示されます
3. 以下を入力：

| 項目 | 必須 | 説明 |
|------|------|------|
| **利用目的** | ○ | ドロップダウンから選択 |
| **その他詳細** | △ | 「その他」を選んだ場合のみ |
| **掲載終了日** | ○ | 明日〜1年以内で指定 |
| **コメント** | - | 承認者への連絡事項（任意） |

#### 利用目的の選択肢
- 自店のホットペッパービューティー掲載ページ
- 自店の公式ホームページおよびブログ
- 自店の公式SNSアカウント
- 自店で使用するチラシ、DM、POPなどの販促物
- その他

### Step 2: 同意書の確認

1. 利用規約を最後までスクロール
2. 10秒間待機（カウントダウン表示）
3. チェックボックスにチェック
4. 「申請する」ボタンをクリック

> **重要**: 同意書を最後まで読み、10秒経過しないと申請ボタンは有効になりません。

### 申請後の流れ
1. 所属長と社長にメールで通知されます
2. 承認されるとメールでお知らせが届きます
3. 7日以内に1回だけダウンロードできます

## 申請履歴の確認

画面上部の「申請履歴」タブをクリックすると、自分の申請一覧が表示されます。

### ステータスの種類

| ステータス | 色 | 説明 |
|------------|------|------|
| **承認待ち** | 黄色 | 管理者の承認を待っています |
| **ダウンロード可** | 緑 | 承認されました。7日以内にダウンロードしてください |
| **DL済み** | 青 | ダウンロード完了 |
| **却下** | 赤 | 申請が却下されました |
| **期限切れ** | グレー | ダウンロード期限が過ぎました |

### 申請のキャンセル
「承認待ち」状態の申請はキャンセルできます。
「キャンセル」ボタンをクリックしてください。

### 詳細の確認
申請をクリックすると詳細情報が表示されます：
- 申請番号
- 画像名
- 利用目的
- 掲載終了日
- 承認者からのコメント
- 却下理由（却下された場合）

## 画像のダウンロード

### ダウンロード手順

1. 「申請履歴」タブを開く
2. 「ダウンロード可」の申請を探す
3. 「ダウンロード」ボタンをクリック
4. 確認ダイアログで「ダウンロード」をクリック
5. 画像がダウンロードされます

### 注意事項

- **ダウンロードは1回のみ**です
- ダウンロード期限は承認から**7日間**です
- ダウンロードした画像には**透かし**が埋め込まれています

### 透かし（ウォーターマーク）について

ダウンロードした画像には、目に見えない透かしが埋め込まれています。
これには以下の情報が含まれます：
- ダウンロードした人の名前
- 承認した人の名前
- 申請番号
- ダウンロード日時

> **警告**: 透かしは画像編集では消せません。不正利用は追跡可能です。

## 掲載終了の確認

掲載終了日を過ぎたら、画像を削除したことを確認する必要があります。

### 確認方法

1. 「申請履歴」タブを開く
2. 期限切れの申請にオレンジ色のバナーが表示されます
3. 「掲載終了を確認」ボタンをクリック
4. 画像を削除したことを確認

### 注意事項
- 確認が完了するまで、毎日リマインダーメールが届きます
- あなたと承認者の両方が確認する必要があります
- 両者が確認するとリマインダーが停止します

## 管理者向け機能

管理者権限を持つユーザーは、画面右上のメニューから管理画面にアクセスできます。

### ダッシュボード
- ユーザー数
- 画像数
- 承認待ち申請数

### ユーザー管理
- ユーザーの追加・編集・削除
- CSVファイルからの一括インポート
- 権限の設定（管理者/一般ユーザー）

### 部門管理
- 部門の追加・編集・削除
- 部門長の設定

### ファイル管理
- 画像のアップロード（ドラッグ＆ドロップ対応）
- フォルダの作成・移動・削除
- 権限の設定（個別/デフォルト）

### 承認申請管理
- **承認待ち**タブ：未処理の申請を処理
- **履歴**タブ：過去の申請を確認
- 掲載終了確認のボタン

### 透かし検証
管理画面の「透かし検証」から、画像に埋め込まれた透かしを確認できます。

## トラブルシューティング

### ログインできない
- 管理者にユーザー登録を依頼してください
- ブラウザのキャッシュをクリアしてみてください

### 画像が表示されない
- フォルダに対する権限があるか確認してください
- 管理者に権限設定を依頼してください

### ダウンロードできない
- 申請が承認されているか確認してください
- ダウンロード期限（7日間）が過ぎていないか確認してください
- すでにダウンロード済みでないか確認してください（1回のみ）

### メールが届かない
- 迷惑メールフォルダを確認してください
- SendGridからのメールを許可してください
